Site Escola e Estudante

O projeto tem como objetivo introduzir o Colégio Estadual Olympia Morais de Tormenta e os integrantes do grupo de alunos do segundo ano D, promovendo a interação através de links para as redes sociais de cada participante. Utilizamos HTML para organizar o conteúdo e CSS para aprimorar a aparência da página, proporcionando uma experiência de navegação amigável.

A estrutura do site é dividida em uma seção de cabeçalho, onde são exibidos o logotipo do projeto e um menu de navegação. Em seguida, temos a seção "Sobre a Escola", com uma breve descrição da história e localização do colégio, complementada por uma imagem representativa. Na seção "Quem são os alunos?", cada estudante é apresentado com sua foto, nome e links para suas redes sociais, permitindo fácil acesso às plataformas como Facebook, Twitter e Instagram. O layout flexível e a meta tag viewport garantem uma exibição responsiva em diferentes dispositivos.

O CSS dá ao site uma aparência moderna e coesa, utilizando variáveis de cores e fontes personalizadas para destacar cada parte do conteúdo. Botões interativos para as redes sociais proporcionam uma experiência dinâmica, incentivando a interação e facilitando a navegação. Este projeto visa criar uma página acessível e atraente, onde o grupo escolar possa ser apresentado de forma profissional e amigável

Site Biblioteca Virtual

Este projeto é uma biblioteca virtual, desenvolvida com HTML, CSS e JavaScript, com o objetivo de proporcionar aos usuários uma plataforma para explorar, pesquisar e adquirir livros online. A página apresenta uma interface limpa e agradável, com uma estrutura fácil de navegar, permitindo que os usuários encontrem livros de diversos gêneros e interesses.

A estrutura do site é construída com HTML, que define o layout e a organização do conteúdo, incluindo um cabeçalho com o logo, menu de navegação e ícones de acesso rápido para favoritos, compras e perfil do usuário. O corpo da página é composto por várias seções que exibem informações sobre os livros, como novos lançamentos, mais vendidos e tópicos populares.

O design do site é responsivo e estilizado com CSS. Utiliza a fonte Poppins do Google Fonts para garantir uma leitura agradável e uma aparência moderna. O layout foi otimizado para ser acessível em dispositivos móveis, utilizando uma barra de pesquisa e sliders interativos para facilitar a navegação. Além disso, o site conta com um carrossel de imagens (usando a biblioteca Swiper) para destacar os lançamentos mais recentes e os livros mais vendidos.

Ao utilizar HTML, CSS e JavaScript, este projeto cria uma experiência interativa e fácil de usar para quem busca novos livros ou deseja explorar mais sobre suas leituras favoritas. O uso do Swiper e a navegação intuitiva garantem que o usuário tenha uma experiência agradável ao explorar o catálogo da biblioteca virtual, enquanto o design limpo e funcional contribui para uma navegação sem complicação.

Site de Assinatura

Este projeto é uma simulação de site de promoção da assinatura Netflix, criada para fornecer informações sobre os benefícios e recursos exclusivos desse plano. O objetivo é incentivar os usuários a assinarem o serviço, destacando vantagens como qualidade de vídeo, dispositivos compatíveis e opções de download para assistir sem conexão à internet.

O site é estruturado com HTML, proporcionando uma organização clara e fácil de navegar, enquanto o design é feito com CSS para garantir uma experiência visual agradável e responsiva. A página é otimizada para dispositivos móveis e desktops, com destaque para imagens chamativas que demonstram as funcionalidades do serviço, como a qualidade de vídeo em 4K e a opção de assistir em diversos dispositivos.

O design da página é moderno, com fontes suaves, e a navegação é simplificada para que os usuários possam acessar rapidamente as informações ou se inscrever no serviço. A página utiliza um esquema de cores e imagens atraentes para destacar a qualidade do conteúdo oferecido pela Netflix, promovendo a conversão de visitantes em assinantes.
